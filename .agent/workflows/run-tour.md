---
description: Menjalankan Project Virtual Tour
---

Untuk menjalankan project ini dan melihat hasilnya di browser:

1. Jalankan perintah instalasi dependensi (jika belum):

```bash
npm install
```

2. Jalankan server development:

```bash
npm run dev
```

3. Buka URL yang muncul di terminal (biasanya http://localhost:5173).

4. Untuk mengganti gambar, letakkan gambar panorama baru di folder `assets/` dan sesuaikan nama file di `src/main.ts`.
